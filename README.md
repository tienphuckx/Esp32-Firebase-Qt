# Esp32-Firebase-Qt

My first experience with ESP32 involved reading real-time data from a DHT11 sensor (temperature, humidity) and GPS, then pushing the data to Firebase and syncing it with a Qt application.

# Devices
- ESP32
- DHT11
- Firebase
- Qt app
